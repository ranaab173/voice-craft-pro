Key Features of This Fully Functional Tool:
Real Text-to-Speech Functionality:

Uses the Web Speech API (built into modern browsers)

No external API keys or dependencies required

Works offline in supported browsers

Multiple Voice Options:

Automatically detects all available voices in your browser

Groups voices by language for easy selection

Both male and female voices available

Complete Audio Controls:

Play, pause, and stop functionality

Adjustable speed, pitch, and volume

Visual progress indicator

Download Capability:

Simulated audio download (in a production environment, you would implement server-side audio generation)

History Feature:

Saves your previous conversions

Allows quick replay of previous texts

Persistent storage using localStorage

Responsive Design:

Works on desktop, tablet, and mobile devices

Clean, modern interface with intuitive controls
